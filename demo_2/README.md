# 📊 Kickstarter Campaign Analysis – Power BI

## 📌 Project Overview

This project analyzes crowdfunding campaigns from **Kickstarter** using **Power BI**.
The objective of this analysis is to explore patterns behind successful and unsuccessful crowdfunding projects and understand how different factors such as category, country, and funding goals influence campaign outcomes.

The dataset was cleaned, merged, and transformed before building an interactive dashboard to visualize key insights.

---

# 🗂 Dataset

The dataset used in this project was obtained from **Kaggle**.

Dataset source:

Kickstarter Projects Dataset
https://www.kaggle.com/datasets/kemical/kickstarter-projects

---

# ⚙️ Data Preparation

Data preparation was performed using **Power Query in Power BI**.

The following steps were applied:

1. Merged the two CSV datasets into a single dataset.
2. Fixed column shifting issues caused by commas inside text fields.
3. Removed rows containing data errors.
4. Removed duplicate records.
5. Cleaned and standardized the dataset.
6. Corrected data types for numeric and date columns.

After these steps, the dataset was ready for analysis and visualization.

---

# 📊 Dashboard

An interactive **Power BI dashboard** was created to analyze Kickstarter campaigns.

The dashboard includes the following KPIs:

* **Total Projects:** 378K+
* **Total Backers:** 40M+
* **Total Pledged:** $2.64 Billion
* **Total Funding Goal:** $18.58 Billion

---

# 📈 Visualizations

The dashboard contains several visualizations:

### Projects by State

A pie chart showing the distribution of projects by status:

* Successful
* Failed
* Canceled
* Suspended
* Live

### Projects by Category

A treemap showing the number of projects in each main category such as:

* Film & Video
* Music
* Publishing
* Technology
* Games
* Design

### Projects Over Time

A line chart showing how the number of projects evolved over the years.

### Projects by Country

A bar chart showing which countries launch the most Kickstarter campaigns.
