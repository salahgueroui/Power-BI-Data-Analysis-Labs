# Power BI Data Analysis Projects

This repository contains a collection of **hands-on Power BI projects and practice demos** created while learning Power BI from a YouTube playlist.

Each project focuses on analyzing a dataset and applying new Power BI concepts such as **data cleaning, data modeling, DAX calculations, and dashboard design**.

The goal of this repository is to document my **learning journey in Data Analytics using Power BI** and build a portfolio of small analytical projects.

---

## What I Learned

During these demos, I practiced several important Power BI skills:

* Importing and connecting data from different sources
* Data cleaning and transformation using **Power Query**
* Creating relationships between tables
* Writing basic **DAX measures**
* Building interactive **data visualizations**
* Designing analytical **dashboards**
* Using filters, slicers, and formatting tools

---
